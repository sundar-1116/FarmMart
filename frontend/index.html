ℹ️ Platform Capabilities & Browser Architecture

About FarmMart

FarmMart is India's premier farm-to-market web application, designed to facilitate a direct connection between local growers and national retail stores. Below is a summary of what this application can do and the browser APIs it leverages.

💾 Client-Side Data Storage (LocalStorage)

The application stores persistent platform data directly inside your browser's LocalStorage. This includes the database of registered farmers, partner retail stores, submitted demands, tasks, and queries, enabling off-grid client-side access.

🔑 Safe Sessions & State Management (sessionStorage)

User logins and active user status are managed securely via sessionStorage. This verifies active sessions within the browser tab and automatically clears authorization data as soon as the tab or window is closed.

🛡️ Native Client Cryptography (Web Crypto API)

To prioritize credentials security, user passwords are not stored in plain text. The application uses the browser's native Web Crypto API to securely hash user passwords using the industry-standard SHA-256 algorithm.

💬 AI-Powered Dashboard Assistant

A responsive, client-side natural language processing helper allows logged-in users to search for crops, match logistics tasks, and get instant answers based on local data stores directly from the portal.
